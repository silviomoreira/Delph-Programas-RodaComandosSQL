# Delph-Programas-RodaComandosSQL
* Objetivo: Ferramenta para executar comandos dml e ddl utilizando originalmente o bde. Ajuda no encontro de erros em scripts sql, demonstrando o resultado
de cada comando sql e seu respectivo erro na aba log.
* Como Operar: Primeiro seleciona-se o Alias no combobox, marcam-se as opções desejadas ou não, e escreve(m)-se o(s) comando(s) na aba [Execute], ou o comando select na aba [Select]
